üìç TECHNICAL LOGIC
--------------------------------------------------------------------------------
FRONTEND (Contact.jsx)
-----------------------
- React functional component with useState hooks
- Framer Motion for animations
- Form state management with controlled inputs
- Cascading dropdown logic: destination resets when package changes
- Available Packages: Premium, Pro, Elite
- Available Destinations: Puri, Bhubaneswar, Chilika, Konark
- API call: POST /api/contact
- Error handling with try-catch and user feedback

BACKEND (Node.js + Express)
---------------------------
Files:
- models/Contact.js      ‚Üí Mongoose schema for storing inquiries
- controllers/contactController.js ‚Üí Business logic + email sending
- routes/contactRoutes.js ‚Üí Route definition (POST /)
- server.js              ‚Üí Route registration (/api/contact)

Flow:
1. Frontend submits JSON ‚Üí POST /api/contact
2. Controller saves data to MongoDB (Contact collection)
3. If EMAIL_USER & EMAIL_PASS exist in .env:
   - Nodemailer sends email to admin with inquiry details
4. Returns success response to frontend

EMAIL INTEGRATION (Nodemailer)
------------------------------
- Service: Gmail (configurable)
- Auth: App Password (not regular password)
- Recipient: Same as EMAIL_USER (admin receives inquiries)
- Content: HTML email with all form fields including package and destination

üìç DATABASE SCHEMA (MongoDB)
--------------------------------------------------------------------------------
Collection: contacts
Fields:
- name: String (required)
- email: String (required, validated)
- phone: String (required)
- message: String (required)
- package: String (default: "General Inquiry") - Premium/Pro/Elite
- destination: String (default: "") - Puri/Bhubaneswar/Chilika/Konark
- status: Enum ["New", "Contacted", "Resolved"] (default: "New")
- createdAt: Date (auto)

üìç ENVIRONMENT VARIABLES REQUIRED
--------------------------------------------------------------------------------
Backend/.env:
- EMAIL_USER=your-gmail@gmail.com
- EMAIL_PASS=xxxx-xxxx-xxxx-xxxx (Gmail App Password)
